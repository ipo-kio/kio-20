Дата-центры (вершинное покрытие графа).
На экране изображен граф (можно наложить на карту России условно).
Он показывает расположение и связь между различными вычислительными центрами.
Часть из центров предполагается оснастить мощными компьютерами для хранения данных так чтобы каждый из оставшихся центров имел прямую связь с одним из них.
Первый критерий: количество дата-центром (меньше - лучше).
Второй параметр: суммарное расстояние центро до ближайших дата-центров.

Усложнение задачи для старших уровней - повышение надежности - дублирование информации - прямая связь с не менее 2 дата-центрами.

Что дано:
Организатор (мы) расставляем на карте датацентры и связываем их каналами (могут быть различные конфигурации для уровней)
Задача игрока: Отметить нужные датацентры, чтоб получился результат.
